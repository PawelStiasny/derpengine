    • Derpengine

    A small shader-based OpenGL scene graph engine.

    Project maintained by PawelStiasny Hosted on GitHub Pages — Theme by mattgraham

    What?

    derpengine compiles and runs on Linux and Windows, uses modern OpenGL and so far implements following features:

    • Extensible scene graph
    • Dynamic shadow maps
    • (Static) environment maps
    • Bumpmapping of unparametrized meshes
    • Heightmap based terrain It is also licensed on the MIT License, so you can do pretty much anything with it.
